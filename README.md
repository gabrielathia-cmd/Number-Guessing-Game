# Number-Guessing-Game
Projeto de estudo em Python com um jogo de adivinhação numérica, criado para praticar lógica de programação, estruturas condicionais, laços de repetição e entrada de dados.
# Jogo de Adivinhação em Python

Este projeto é um **exercício de estudo** em Python, com o objetivo de praticar conceitos básicos da linguagem, como:

- entrada de dados com `input()`
- estruturas condicionais (`if`, `elif`, `else`)
- laços de repetição (`while`)
- uso de variáveis
- importação de bibliotecas, como `random`

## Sobre o projeto

O programa gera um número aleatório entre **1 e 100** e pede para o jogador escolher um nível de dificuldade:

- **1 - Fácil** → 20 chances
- **2 - Médio** → 10 chances
- **3 - Difícil** → 5 chances

Depois disso, o jogador deve tentar adivinhar o número sorteado.

## Objetivo

O objetivo deste código é servir como **material de aprendizagem**, ajudando a entender a lógica de programação de forma simples e prática.
